Project Title

A Research on the relationship between the seriousness of Natural Disaster and the Sentiment of Twitter Text - A COVID-19 Pandemic Case Study

Project Objectives

  • Study the social media’s text data structures
  • Apply basic Natural Language Processing Technique to perform sentiment analysis on tweet text
  • Study if there is association between the level of natural disaster and sentiment of the text

Programming Languages

  • Python to connect to Twitter API and obtain raw tweet text
  • R to data wrangling, presentation and statistical analysis

Cities to Study

  • San Francisco, CA
  • Los Angeles, CA
  • Chicago, IL
  • Miami, FL
  • New York, NY

Significant population in the country with higher COVID-19 Confirmed Rate

COVID-19 Data

  • San Francisco: I used San Francisco County, CA data
  • Los Angeles, I used Los Angeles County, CA data.
  • Chicago, I used Cook County, Illinois data.
  • Miami, I used Miami-Dade County, Florida data.
  • New York, I used the combined 5 counties data, and the counties are:
    • Brooklyn, NY
    • Bronx. NY
    • Manhattan, NY
    • Queens, NY
    • Staten Island, NY

Steps:

  • Connect to Twitter API using Python and gather data
  • Load original csv file to R for data wrangling
    • Remove Hyperlinks, Punctuation, Hashtags etc
    • Remove stop words
  • Use packages in R for Data Visualization and Summary Statistics

Comments:

  • Since data for five cities is not normally distributed. We can not use traditional ANOVA. -(P-value > 0.05 in Shapiro-Wilk normality test)
  • I use Wilcoxon rank sum test to test the difference between 5 cities
    • Most of the cities doesn’t have significance difference.
    • i.e. P-value > 0.05
